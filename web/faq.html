Frequently Asked Questions

LLM Profiles - Structured Data Profiles for AI & SEO

Note: This is a reference page. For complete documentation, visit https://llmprofiles.org/
What is LLM Profiles?
LLM Profiles provides opinionated, testable structured data profiles that bridge the gap between SEO and AI applications. Each profile includes machine-enforceable schemas, training data exports, and AEO-optimized guidelines.
How do I get started with LLM Profiles?
  1. Browse available profiles at https://llmprofiles.org/
  2. Choose a profile that matches your content type (FAQPage, Article, Course, etc.)
  3. Implement the JSON-LD markup following the profile guidelines
  4. Validate your implementation using the provided schemas
  5. Export training data for AI/ML applications
What profiles are available?
Currently available profiles include: FAQPage, Article, Course, Event, ProductOffer, QAPage, JobPosting, LocalBusiness, SoftwareApplication, and Review. Each profile is versioned and includes schemas, examples, and training data.
How do I validate my structured data?
Each profile includes JSON schemas for validation:
  • page.schema.json - Validates your JSON-LD markup
  • output.schema.json - Validates extracted content for RAG/AI
Use these in CI/CD to ensure compliance before deployment.
What is AEO (Answer Engine Optimization)?
AEO is the practice of optimizing structured data for AI-powered answer engines. LLM Profiles include built-in AEO features like stable IDs, language hints, disambiguation links, and evidence anchors to improve AI retrieval and answer quality.
How do I discover available profiles programmatically?
Use the discovery endpoints:
  • /.well-known/llmprofiles.json - Machine-readable discovery
  • /api/discovery.json - API discovery endpoint
  • /index.json - Profile registry
Can I contribute new profiles?
Yes! LLM Profiles is open source. You can contribute new profiles by following the contribution guidelines in the GitHub repository. Each profile should include schemas, examples, and training data.
How do I export training data?
Each profile includes a training.jsonl file that defines the format for exporting your content. Generate training data that mirrors your on-page semantics for use in RAG pipelines and AI fine-tuning.
